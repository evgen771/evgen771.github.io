 **LibreOffice доступен в коллекции портов FreeBSD.** 

По умолчанию пакет приходит только на английском языке, для локализованной версии необходимо установить пакет языка. 

### Установка

Есть два основных метода установки LibreOffice на FreeBSD: 

#### Через менеджер пакетов pkg 

Нужно открыть терминал и выполнить команду:

`sudo pkg install libreoffice`
    
Система разрешит зависимости и предложит подтверждение. Нужно ввести «y» и нажать Enter. 

После установки проверить версию:

`libreoffice --version`

#### Из портов

Для пользователей, которые предпочитают пользовательские варианты компиляции, есть альтернативный метод:

Перейти в каталог порта LibreOffice:

`cd /usr/ports/editors/libreoffice`

Собрать и установить LibreOffice:

`sudo make install clean`

Этот процесс может занять некоторое время, в зависимости от характеристик системы.

Перед установкой необходимо обновить систему FreeBSD и убедиться, что установлены необходимые зависимости. 

Большинство необходимых библиотек будут установлены автоматически, но может потребоваться X11 (если используется графический интерфейс) и некоторые дополнительные шрифты. 

#### Некоторые настройки LibreOffice на FreeBSD:

**Установка языка по умолчанию.** 

Нужно открыть LibreOffice, перейти в «Инструменты» → «Параметры», в разделе «Настройки языка» → «Языки» установить желаемый язык, нажать «ОК» и перезапустить LibreOffice.

или:

`# pkg ins -y ru-libreoffice-25.8.1.1`

#### включение поддержки Java 

Некоторые функции LibreOffice требуют Java. 

Если нужно, установить Kit для разработки Java (JDK). 

- Настроить Java в LibreOffice: 

открыть LibreOffice, перейти в «Инструменты» → «Параметры» → «Дополнительно», проверить «Использовать среду выполнения Java» и выбрать установленный JDK, нажать «ОК» и перезапустить LibreOffice.

#### Решение проблем

Некоторые проблемы, которые могут возникнуть при использовании LibreOffice на FreeBSD, и способы их решения:

#### LibreOffice не запускается. 

- Можно запустить программу в безопасном режиме:

`libreoffice --safe-mode`

- Это отключает расширения и сбрасывает настройки пользователя.

#### Не хватает шрифтов. 

В этом случае нужно установить дополнительные шрифты:

`sudo pkg install fonts-noto`

#### Медленная работа. 

- Можно отключить аппаратное ускорение: 

открыть LibreOffice, перейти в «Инструменты» → «Параметры» → «Вид», снять галочку «Использовать аппаратное ускорение» и перезапустить LibreOffice.
