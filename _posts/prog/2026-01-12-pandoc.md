#### Установка Pandoc

```sh
# pkg search pandoc
hs-pandoc-3.8.1_1              Conversion between markup formats
hs-pandoc-crossref-0.3.18.2    Pandoc filter for numbering figures and tables
py311-pandoc-plantuml-filter-0.1.5 Pandoc filter which converts PlantUML code blocks to PlantUML images
py311-pandocfilters-1.5.1_1    Utilities for writing pandoc filters in python
py311-pypandoc-1.14_1          Thin wrapper for pandoc
# pkg install hs-pandoc-3.8.1_1
```

#### Такой метод конвертации:

`pandoc -o sample.html sample.md`

`pandoc -s -o sample.html sample.txt`

`pandoc -o sample.odt sample.md`

`pandoc -o sample.txt sample.md`

`pandoc -f markdown -t latex -s -o sample.tex sample.md`

*-o (out) выходной формат файла ( -f (от) и -t (для) варианты используются, чтобы сказать pandoc какие форматы файлов вы хотите конвертировать из и в )*

#### Для создания PDF-файла укажите выходной файл с .pdf-расширением:

>Нужен установленный `LaTeX`

`pandoc test.txt -o test.pdf`

    or

`pandoc test.txt -V lang=RU-ru -o test.pdf`

***или лучше так,с указанием шрифта***:

`pandoc --pdf-engine=xelatex -V 'mainfont:Arial' -t beamer test.txt -o test.pdf`

*(нужен установленный шрифт **Arial**)*

*По умолчанию pandoc будет использовать LaTeX для создания PDF, для чего требуется установить механизм LaTeX*    

#### Для преобразования hello.html из HTML в Markdown:

`pandoc -f html -t markdown hello.html`
