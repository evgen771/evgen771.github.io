В `FreeBSD` для просмотра зависимостей пакетов при обновлении или до удаления пакета используются команда `pkg info` и утилита `pkg tree`. 

#### Команда pkg info

Позволяет показать зависимости пакета:

- Для этого нужно использовать параметр -d. 

*Например, команда:*

`pkg info -d python`

отобразит зависимости пакета Python.

*Некоторые особенности использования:*

- Ключ -r показывает пакеты, от которых зависит указанный пакет. Например, команда

`pkg info -rx curl`

выводит, что Curl зависит от пакета ca_root_nss.

- Ключ -R позволяет показать зависимости от указанного пакета. 

Например, команда:

`pkg info -Rx curl`

показывает, что от Curl зависят другие пакеты

- Ключ -x позволяет использовать регулярные выражения, что позволяет не указывать полное название пакета.

*Важно:* 

команда ищет зависимости только среди установленных пакетов, а не во всём репозитории. 

#### Утилита pkg tree

Генерирует графическое дерево пакетов и их зависимостей. 

Чтобы показать зависимости всех пакетов, утилиту нужно запустить без аргументов. 

Чтобы отобразить зависимости одного пакета, запустить с аргументом. 

*Некоторые особенности использования:*

- Параметр -t показывает только пакеты верхнего уровня, -b — только пакеты нижнего уровня

- Параметр -v позволяет показать зависимости зависимостей

- Вместе параметры -t и -q дают список пакетов верхнего уровня без их зависимостей

*Важно:*

порты, которые имеют обновлённые порты в своих зависимостях, не показывают зависимости, если старый порт удалён — в этом случае утилита выводит «(unknown)»
