**Xpdf** — свободная программа для просмотра PDF для X Window System и Motif.

**Xpdf** может быть запущен на многих Unix-подобных системах.

Xpdf включает в себя несколько программ, для работы которых не требуется X Window System, в том числе программы, позволяющие извлекать изображения из файлов PDF или конвертировать PDF в PostScript или текст. 

### Программа Xpdf доступна на следующих сайтах:

- **Официальный сайт** — xpdfreader.com.
- **Репозитории** с версиями программы, например, в ALT Linux — xpdf-4.06-alt1.

### Запускать в терминале

Пример:

`$ ~ pdftotext  *.pdf `

### В состав Xpdf входит целый набор утилит:

---

```
xpdf/XpdfReader — основная программа для просмотра PDF-файлов
Для запуска xpdf просто введите:
xpdf *.pdf
```
---

```
pdftotext — утилита для конвертации PDF в обычный текст
Синтаксис:
pdftotext [опции] pdffile textfile
Пример — конвертировать text.pdf в text.txt:
pdftotext text.pdf text.txt
Если не указан выходной файл, pdftotext создаст файл с тем же именем и расширением .txt (text.txt)
Некоторые полезные флаги:
-f N
 начать с страницы N
-l N
закончить на странице N
-layout
сохранить базовое расположение текста (столбцы, переносы)
or
pdftotext file.pdf
```

---

```
pdftops — утилита для конвертации PDF в PostScript
Для создания файла PostScript запустите pdftops:
pdftops file.pdf
```

---

```
pdftopng — утилита для конвертации PDF страниц в PNG
pdftopng [options] *.pdf  *.png
pdftopng -f* -l* *.pdf  *.png
-f число
Указывает первую страницу для конвертации. 
-l число
Указывает последнюю страницу для конвертации. 
-mono
Создайте монохромное изображение (вместо цветного). 
-gray
Создайте изображение в оттенках серого (вместо цветного изображения).
```
---

```
pdftohtml — утилита для конвертации PDF в HTML-формат
Преобразование определённых страниц PDF в HTML — используются опции -f и -l. 
Например, чтобы сгенерировать HTML-код PDF-документа со страниц 2–3, используется команда
pdftohtml -f 2 -l 3.pdf
Преобразование PDF-файла в одностраничный HTML — используется опция -s. По умолчанию           программа преобразует PDF в HTML на нескольких страницах.
Игнорирование изображений во время преобразования — используется опция -i. 
Это полезно для   документов с большим количеством текста, где изображения не важны (например, юридических документов)
Преобразование PDF-файла в HTML с другим уровнем масштабирования — используется опция -zoom. Например, команда
pdftohtml -zoom 2.0.pdf
преобразует PDF-файл в HTML-файл с уровнем масштабирования 200%.
```
---

### Существует несколько дополнительных утилит (которые подробно описаны в их справочных страницах):

- pdftoppm — утилита для конвертации PDF страниц в файлы изображений PPM/PGM/PBM
- pdfinfo — утилита для выборки мета-данных
- pdfimages — утилита для извлечения изображений из PDF файлов
- pdffonts — выводит список шрифтов, используемых в PDF файле
- pdfdetach — извлекает прикрепленные файлы из PDF файлов
