title: gpart - утилита для управления разделами
categories: Программы
---

- gpart show - просмотр конфигурации всех дисков.

- gpart destroy -F /dev/da1 - удаление таблицы и всех разделов.

(без ключа -F таблица удалена не будет если остались разделы)

- gpart delete -i 2 da1 - удаление раздела (2 раздел на диске da1)

- gpart create -s GPT /dev/da1 - создание таблицы разделов.

Допустим, что USB флешка определяется в устройствах как `/dev/da0`. 

Тогда для форматирования накопителя в файловую систему `UFS2` с таблицей разделов в формате `GPT` нужно выполнить в терминале следующие команды:

```sh
gpart destroy -F /dev/da0
gpart create -s gpt /dev/da0
gpart add -t freebsd-ufs /dev/da0
newfs /dev/da0p1
```

Если же необходимо отформатировать накопитель в файловую систему `FAT32` с таблицей разделов в формате `MBR`, то нужно выполнить в терминале другие команды:

```sh
gpart destroy -F /dev/da0
gpart create -s mbr /dev/da0
gpart add -t fat32 /dev/da0
newfs_msdos -F 32 /dev/da0s1
```
