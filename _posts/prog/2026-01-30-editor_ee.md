Во FreeBSD можно настроить использование редактора ee (easy editor) по умолчанию для конкретного пользователя. 

**По умолчанию в системе используется редактор vi**

#### Настройка зависит от оболочки (shell). 

Основные конфигурационные файлы, связанные с профилем пользователя, находятся в домашнем каталоге пользователя. 

#### В csh

Чтобы сменить редактор по умолчанию в оболочке csh, нужно: 

- Залогиниться под пользователем
- Перейти в домашнюю папку
- Открыть и отредактировать файл `ee .cshrc`
- Найти строку `setenv EDITOR vi` и поменять её на `setenv EDITOR ee`
- Нажать кнопки `Esc+Enter` и сохранить файл при выходе, выбрав опцию `a) save changes`
- Перезапустить компьютер

Теперь `ee` будет всегда использоваться при вводе команд, требующих вызова консольного текстового редактора. 

#### В bash

Чтобы изменить редактор по умолчанию в оболочке bash, нужно: 

Создать файл: 

`~/.bash_profile (по умолчанию он не существует)`

Добавить параметры:

`export EDITOR=ee`

Сохранить и закрыть файл (нажать `Ctrl + C`, ввести `exit`, а затем нажать клавишу `Enter`)

Чтобы немедленно активировать изменения, ввести:

`source ~/.bash_profile`
