**В ImageMagick есть встроенные (встроенные) изображения**, которые можно использовать как файлы изображений. Для этого используется тег формата 

magick:. 

Например, для запроса встроенного изображения можно использовать синтаксис magick:name (например, magick:logo).

Также для обратной совместимости доступны спецификации изображений с именами, например, GRANITE:, LOGO:, NETSCAPE:, ROSE:

#### Принцип работы

Встроенные изображения доступны для обработки, как и любые другие изображения, но **имя файла не указывается**. `ImageMagick` не может автоматически определить формат изображения, поэтому необходимо явно указать его.

#### Примеры

Некоторые встроенные изображения в ImageMagick:

- **GRANITE** — узор из плиток текстур гранита размером 128×128 пикселей;
- **LOGO** — логотип ImageMagick, размер 640×480 пикселей;
- **NETSCAPE** — изображение с цветами в формате Netscape, размер 216×144 пикселей.
---

#### gradient*

Имена файлов с градиентом имеют вид `gradient:color1-color2`. Эти изображения создаются путем постепенного изменения цвета от `color1` по верхнему краю до `color2` по нижнему. Не путайте этот формат изображения с классом [GradientFill](https://rmagick.github.io/struct.html#GradientFill), который является частью `RMagick`.

#### granite

>Пестрое серое изображение, подходящее для использования в качестве текстуры плиточного фона.

Открыть:

`display granite:`

Открыть:

#### pattern

- Этот формат содержит ряд встроенных шаблонов, на которые можно ссылаться, указав название шаблона. Например, `pattern:checkerboard. 
- Список допустимых шаблонов смотрите на [этой странице.](https://www.imagemagick.org/script/formats.php) 
- Если вы не укажете размер, будет использован размер шаблона по умолчанию. 
- Если вы укажете размер, шаблон будет повторяться по мере необходимости для заполнения изображения.
- Для использования встроенного изображения можно указать его в команде, например: 

```
magick -size 640x480 pattern:checkerboard checkerboard.png
magick logo:logo imagemagick.png (640x480) из коробки размер
magick -size 640x480 logo:logo imagemagick.png  из коробки размер
magick logo:logo -resize 800x600 imagemagick.png (важен порядок вставки опций!!!)
```

#### plasma

Создает причудливое психоделическое изображение. Для достижения наилучших результатов укажите пару цветов в имени файла (`plasma:red-blue`) или укажите имя файла `plasma:fractal` для достижения лучших результатов.

#### rose

>Маленькая картинка с изображением розы

Открыть:

`display rose:`

#### xc*

Укажите [название цвета](https://rmagick.github.io/imusage.html#color_names) после префикса `xc:`. Например, `xc:red`. 

- Этот формат представляет собой просто изображение указанного цвета. 
- Вы можете получить точно такие же результаты, указав цвет фона при создании изображения.
