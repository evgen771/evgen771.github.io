## Пример

- pdfseparate -f 3 Pdfarranger.pdf Pdfarranger-%d.pdf
- pdfseparate -f 3 -l 5 Pdfarranger.pdf Pdfarranger-%d.pdf

Команда pdfseparate из пакета утилит poppler-utils извлекает определенные страницы из исходного PDF-файла и сохраняет каждую из них как отдельный документ.
Введенная вами команда выполнит следующее:

* -f 3 (First) — указывает начать извлечение с 3-й страницы.
* -l 5 (Last) — указывает закончить извлечение на 5-й странице (включительно).
* Pdfarranger.pdf — имя вашего исходного (входного) многостраничного файла.
* Pdfarranger-%d.pdf — шаблон имени для выходных файлов. Конструкция %d является обязательной: она автоматически заменяется на номер извлеченной страницы.

### Результат выполнения

В текущей папке будут созданы 3 новых одностраничных PDF-файла:

   1. Pdfarranger-3.pdf (содержит только 3-ю страницу оригинала)
   2. Pdfarranger-4.pdf (содержит только 4-ю страницу оригинала)
   3. Pdfarranger-5.pdf (содержит только 5-ю страницу оригинала)


### Альтернатива: Вырезка диапазона в один файл

Если ваша задача — не разделить страницы поодиночке, а вырезать диапазон с 3 по 5 страницу в один общий файл, воспользуйтесь родственной утилитой `pdfimages` или `pdftoppm`, но лучше всего для этого подойдет `pdfjam` или `pdftk`.

Быстрее и проще всего это сделать утилитой `pdfseparate` не умеет объединять, поэтому для склейки в один файл используют `pdfunite`:

- pdfseparate -f 3 -l 5 Pdfarranger.pdf temp-%d.pdf
- pdfunite temp-3.pdf temp-4.pdf temp-5.pdf output.pdf
- rm temp-*.pdf

Или одной командой через `pdftk` (если он установлен):

- pdftk Pdfarranger.pdf cat 3-5 output Pdfarranger_3-5.pdf
