В терминале ввести:
```
display
```
Меню Short Cuts => правая кнопка => выбрать Open =>

Появится окно.

Нажать Grab =>

Новое окно.

Указать задержку, после которой создается снимок =>

Опять нажать Grab

Курсор изменяет форму. Обвести область на экране и щелчком получить ее снимок.

Двойной щелчок на окне дает снимок этого окна.

Из контекстного меню display:

Сохранение снимка - Save

В открывшемся окне - Format.

Выбрать формат изображения: в списке почти сотня поддерживаемых форматов.
