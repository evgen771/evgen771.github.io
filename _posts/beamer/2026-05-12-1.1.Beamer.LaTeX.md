`Beamer` — это класс для `LaTeX`, предназначенный для создания презентаций. 

Он позволяет оформлять слайды с математическими формулами, иллюстрациями, анимацией и другими элементами.

### Основные возможности

- Создание структурированных презентаций (разделы, подразделы, списки).
- Включение сложных математических формул (полная поддержка LaTeX).
- Настройка внешнего вида: темы, цвета, шрифты, стили маркеров.
- Анимация и последовательное появление элементов на слайде.
- Гиперссылки и кнопки навигации.
- Поддержка мультимедиа (видео, звук).
- Экспорт в PDF (кросс‑платформенный формат).
- Возможность вывода на два экрана (например, проектор + монитор докладчика).

### Базовая структура документа

```latex
\documentclass[10pt,pdf,hyperref={unicode}]{beamer}
\usetheme{тема}                    % выбор темы оформления
\useinnertheme{тема}             % стиль внутренних элементов
\useoutertheme{тема}            % стиль внешних элементов

\begin{document}

\begin{frame}
\frametitle{Заголовок слайда}
% Содержание слайда
\end{frame}

\end{document}
```

### Ключевые команды

1. **`\documentclass{beamer}`** — указание класса документа.  
   Опции:  
   - `10pt` — размер шрифта;  
   - `pdf` — вывод в PDF;  
   - `hyperref={unicode}` — поддержка Unicode для гиперссылок.

2. **`\usetheme{...}`** — выбор темы оформления (например, `Berlin`, `Darmstadt`).

3. **`\useinnertheme{...}`** и **`\useoutertheme{...}`** — настройка внутренних и внешних элементов.


4. **`\begin{frame}`** и **`\end{frame}`** — блок слайда.  
   - **`\frametitle{...}`** — заголовок слайда.

5. **`\setbeamercovered{transparent}`** — полупрозрачное отображение скрытых элементов при анимации.

6. **`\includemovie`** и **`\sound`** — вставка видео и звука.

7. **`\beamerbutton{...}`**, **`\beamergotobutton{...}`** — создание кнопок навигации.

8. **`\label{...}`** и гиперссылки — организация переходов между слайдами.

9. **`\setbeameroption{second mode text on second screen=...}`** — настройка второго экрана.

### Примеры тем оформления

Beamer включает множество встроенных тем:

  `Antibes`, `Berlin`, `Boadilla`, `Darmstadt`, `Dresden`, `Frankfurt`, `Goettingen`, `Hannover`, `Montpellier`, `Pittsburgh`, `Singapore` и др.


### Советы по использованию

1. **Пропорции слайдов**: 

   - Стандарт — `4:3` 
   - Для `16:9` укажите `aspectratio=169` в опциях `\documentclass`.

2. **Цветовая настройка**: 
  
   ```latex
   \setbeamercolor{normal text}{bg=белый,fg=чёрный}
   ```

3. **Анимация**:  
 
   - Используйте `<номер>` для последовательного появления элементов:
    
     ```latex
     \item<1-> Первый пункт
     \item<2-> Второй пункт
     ```

4. **Библиография**:  
 
   - Включите в слайд через окружение `thebibliography` или используйте BibTeX.

### Вывод

Результат компиляции — `PDF‑файл`, совместимый с любыми устройствами (проекторы, планшеты, ПК). 

`Beamer` особенно удобен для научных презентаций благодаря мощной поддержке формул и структурированности.

**Примечание**: для компиляции используйте `pdflatex`, `lualatex` или `xelatex`.
