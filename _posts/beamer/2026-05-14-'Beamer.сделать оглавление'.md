В LaTeX оглавление создается автоматически одной командой.

Компилятор сам собирает названия разделов (`\section`, `\subsection`) и номера страниц, на которых они расположены.

## Основная команда

Вставьте команду `\tableofcontents` в то место документа, где должно быть оглавление (обычно сразу после титульного листа или аннотации):

```latex
\documentclass{article}
\usepackage[T2A]{fontenc}
\usepackage[utf8]{inputenc}
\usepackage[russian]{babel}

\begin{document}

\title{Мой документ}
\maketitle

\tableofcontents % Сюда будет вставлено оглавление
\newpage

\section{Введение}
Текст введения...

\section{Основная часть}
\subsection{Постановка задачи}
Текст подраздела...

\end{document}
```

Важное правило: 

Чтобы оглавление отобразилось и обновилось, документ нужно скомпилировать минимум 2 раза. 

При первом проходе `LaTeX` собирает заголовки во вспомогательный файл (`.toc`), а при втором — переносит их на страницу.

## Оглавление в презентациях Beamer

Поскольку вы верстаете презентацию, в классе `beamer` оглавление обычно выносят на отдельный слайд (`фрейм`) в начале выступления:

```latex
\begin{frame}{Содержание презентации}
    \tableofcontents
\end{frame}
```

## Полезные фишки для Beamer:

* Скрытие мелких подразделов: Чтобы не перегружать слайд, можно скрыть подразделы, оставив только главные разделы:

`\tableofcontents[hideallsubsections]`

* Подсветка текущего раздела: 

Если вы хотите показывать оглавление в начале каждого нового раздела с подсветкой текущей темы, добавьте в преамбулу:

```latex
\AtBeginSection[]
{
  \begin{frame}{План выступления}
    \tableofcontents[currentsection]
  \end{frame}
}
```

## Как управлять элементами оглавления

* Исключение заголовка из оглавления: 

Если заголовок нужен в тексте, но его не должно быть в содержании, добавьте звездочку `*`:

`\section*{Введение без номера} % Не попадет в оглавление`

* Принудительное добавление: 

Если вы хотите добавить в оглавление неномерованный раздел (со звездочкой), используйте команду сразу под ним:

```latex
\section*{Введение}
\addcontentsline{toc}{section}{Введение}
```

* Глубина оглавления: 

По умолчанию в оглавление включаются разделы до `3-го` уровня (`subsubsection`). 

Изменить это можно в преамбуле:

`\setcounter{tocdepth}{2} % Показывать только section и subsection`
