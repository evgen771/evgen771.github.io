### Что такое Beamer


`Beamer` — класс `LaTeX` для создания профессиональных презентаций. 

Позволяет:

* оформлять слайды с математическими формулами;
* использовать анимации и переходы;
* настраивать внешний вид (темы, цвета, шрифты);
* вставлять изображения, таблицы, списки;
* создавать гиперссылки и кнопки навигации;
* экспортировать в PDF.

### Установка и подготовка

Убедитесь, что у вас установлен `LaTeX‑дистрибутив` (`TeX Live`, `MiKTeX` и др.). 

`Beamer` обычно входит в стандартный набор пакетов.

### Базовая структура документа


Создайте файл `.tex` с таким содержимым:

```latex
\documentclass[10pt,pdf,hyperref={unicode}]{beamer}
\usetheme{Madrid}                % выбор темы
\usecolortheme{beaver}         % цветовая схема
\usefonttheme{serif}           % шрифт


% Информация для титульного слайда
\title{Моя презентация}
\author{Ваше имя}
\institute{Организация}
\date{\today}

\begin{document}

% Титульный слайд
\begin{frame}
\titlepage
\end{frame}

% Обычный слайд
\begin{frame}
\frametitle{Заголовок слайда}
Содержание слайда...
\end{frame}

\end{document}
```

### Основные команды и их назначение

* **`\documentclass[опции]{beamer}`** — объявление класса документа.  
  * `10pt` — размер шрифта;  
  * `pdf` — вывод в PDF;  
  * `hyperref={unicode}` — поддержка Unicode для ссылок.

* **`\usetheme{...}`** — выбор темы оформления (например, `Madrid`, `Berlin`).

* **`\usecolortheme{...}`** — цветовая схема (например, `beaver`, `dolphin`).

* **`\usefonttheme{...}`** — стиль шрифта (например, `serif`, `structurebold`).

* **`\title{...}`, `\author{...}`, `\institute{...}`, `\date{...}`** — данные для титульного слайда.

* **`\begin{frame}`** и **`\end{frame}`** — блок одного слайда.

* **`\titlepage`** — автоматически создаёт титульный слайд.

* **`\frametitle{...}`** — заголовок текущего слайда.

### Создание слайдов

#### Простой слайд
```latex
\begin{frame}
\frametitle{Введение}
Это первый слайд презентации.
\end{frame}
```

#### Слайд с списком
```latex
\begin{frame}
\frametitle{Основные пункты}
\begin{itemize}
    \item Пункт 1
    \item Пункт 2
    \item Пункт 3
\end{itemize}
\end{frame}
```

#### Слайд с нумерованным списком
```latex
\begin{frame}
\frametitle{Шаги выполнения}
\begin{enumerate}
    \item Шаг 1
    \item Шаг 2
    \item Шаг 3
\end{enumerate}
\end{frame}
```

### Анимация и последовательное появление элементов

Используйте `<номер>` для контроля появления элементов:

```latex
\begin{frame}
\frametitle{Анимация}
\begin{itemize}
    \item<1-> Первый пункт (появится на 1‑м шаге)
    \item<2-> Второй пункт (появится на 2‑м шаге)
    \item<3-> Третий пункт (появится на 3‑м шаге)
\end{itemize}
\end{frame}
```

Для полупрозрачного отображения скрытых элементов добавьте:

```latex
\setbeamercovered{transparent}
```

### Вставки изображений

```latex
\begin{frame}
\frametitle{Изображение}
\begin{figure}
    \centering
    \includegraphics[width=0.5\textwidth]{example-image}
    \caption{Подпись к изображению}
\end{figure}
\end{frame}
```

* `width=0.5\textwidth` — ширина изображения (50 % от ширины слайда);
* `example-image` — имя файла (без расширения, если формат поддерживается).

### Темы оформления

Beamer включает множество встроенных тем. 

Ниже — примеры популярных:

#### Основные темы (`\usetheme{...}`)

* `Madrid` — сбалансированный дизайн, подходит для большинства презентаций;
* `Berlin` — строгий, с вертикальной полосой слева;
* `Darmstadt` — классический, с колонтитулами;
* `Pittsburgh` — минималистичный, без лишних элементов;
* `Singapore` — яркий, с цветными блоками;
* `Antibes` — лёгкий, с градиентами;
* `Frankfurt` — формальный, с рамками.


#### Цветовые схемы (`\usecolortheme{...}`)

* `beaver` — коричневые тона;
* `dolphin` — синие и серые оттенки;
* `orchid` — фиолетовые акценты;
* `rose` — розовые и красные тона;
* `seagull` — нейтральные серые и белые цвета.

#### Шрифтовые темы (`\usefonttheme{...}`)

* `serif` — шрифты с засечками (классический вид);
* `structurebold` — заголовки жирным шрифтом;
* `structuresmallcapsserif` — заголовки прописными буквами;
* `default` — стандартные шрифты LaTeX.


### Настройка внешнего вида

#### Изменение цветов элементов
```latex
\setbeamercolor{normal text}{fg=black,bg=white}      % текст и фон
\setbeamercolor{title}{fg=blue}                       % цвет заголовка
\setbeamercolor{frametitle}{fg=red}                 % цвет заголовка слайда
```

#### Размер шрифта

Укажите в опциях класса:

```latex
\documentclass[12pt]{beamer}  % 12 пунктов
```
Доступные размеры: `8pt`, `9pt`, `10pt`, `11pt`, `12pt`, `14pt`, `17pt`, `20pt`.

#### Пропорции слайда

Для формата 16:9 добавьте в опции класса:
```latex
\documentclass[aspectratio=169]{beamer}
```

### Гиперссылки и навигация

#### Внутренняя ссылка

```latex
\label{slide:example}  % метка слайда

% В другом месте презентации:
\hyperlink{slide:example}{Перейти к примеру}
```

#### Кнопка навигации

```latex
\beamerbutton{Далее}
\beamergotobutton{Вперёд}
\beamerskipbutton{Пропустить}
\beamerreturnbutton{Назад}
```

### Библиография

Разместите на слайде:

```latex
\begin{frame}
\frametitle{Литература}
\begin{thebibliography}{99}
    \bibitem{key1} Автор. Название. Год.
    \bibitem{key2} Другой автор. Другое название. Год.
\end{thebibliography}
\end{frame}
```

Или используйте `BibTeX` (вне документа).

### Компиляция и вывод

1. Сохраните файл как `presentation.tex`.
2. Скомпилируйте с помощью:  
   * `pdflatex presentation.tex` (стандартный способ);  
   * `lualatex presentation.tex` (для сложных шрифтов);  
   * `xelatex presentation.tex` (для Unicode).
3. Получите файл `presentation.pdf`.

### Советы для начинающих

1. **Начинайте с простой темы** (например, `Madrid`).
2. **Используйте шаблоны** — копируйте рабочие примеры.
3. **Тестируйте компиляцию** после каждого изменения.
4. **Ограничьте текст на слайде** — лучше меньше, но понятнее.
5. **Добавляйте изображения** для наглядности.
6. **Проверяйте шрифты** — некоторые могут не поддерживаться.
7. **Используйте анимации умеренно** — они должны помогать, а не отвлекать.
