Для проигрывания музыкального файла с помощью инструментов `FFmpeg` чаще всего используется утилита `ffplay`. Хотя сам `ffmpeg` предназначен в первую очередь для конвертации, он также может выводить звук напрямую через аудиодрайверы системы. 

### 1. Самый простой способ (через ffplay)

Утилита `ffplay` — это легкий медиаплеер, входящий в пакет `FFmpeg`. 

Для запуска файла просто введите его имя:

`ffplay audio.mp3`

* Клавиши управления: Space или p — пауза, 9/0 — громкость, Стрелки — перемотка.
* Режим без окна: Чтобы проиграть звук в фоне без открытия графического окна (только в терминале), используйте:

`ffplay -nodisp -autoexit audio.mp3`

* -nodisp: отключает визуализацию.
* -autoexit: автоматически закрывает плеер по завершении трека.


### 2. Через ffmpeg (прямой вывод)

Сам ffmpeg может проиграть файл, если направить вывод в виртуальное устройство воспроизведения (например, SDL, ALSA или dshow).

* Linux (через ALSA):

`ffmpeg -i audio.mp3 -f alsa default`

* Windows (через SDL):

`ffmpeg -i audio.mp3 -f sdl "default"`

(Примечание: для этого FFmpeg должен быть собран с поддержкой соответствующих библиотек). 

### 3. Проигрывание в цикле

Если нужно зациклить воспроизведение файла:

`ffplay -loop 0 audio.mp3`

0 означает бесконечный повтор, любое другое число — количество повторений.

---

Утилита ffplay по умолчанию поддерживает только один файл на входе и не имеет встроенного механизма для работы с каталогами или плейлистами. Однако это можно реализовать с помощью консольных команд или создания текстового файла-плейлиста. 

### Способ 1: Использование циклов в терминале

Этот метод заставляет систему поочередно запускать `ffplay` для каждого файла в папке. Важно добавлять флаг `-autoexit`, чтобы плеер закрывался сам по завершении трека и переходил к следующему.

* Linux/macOS (Bash):

`for f in *.mp3; do ffplay -nodisp -autoexit "$f"; done`

* Windows (PowerShell):

`foreach ($f in Get-ChildItem -Filter *.mp3) { ffplay -nodisp -autoexit $f.Name }`

* Windows (CMD):

`for %f in (*.mp3) do ffplay -nodisp -autoexit "%f"`


### Способ 2: Использование demuxer concat (плейлист)

Этот способ позволяет ffplay воспринимать список файлов как один непрерывный поток, что обеспечивает более плавный переход между треками.

1. Создайте текстовый файл `playlist.txt` со списком файлов в таком формате:

```bash  
file 'track1.mp3'
file 'track2.mp3'
```

2. Запустите проигрывание командой:
   
`ffplay -f concat -safe 0 -i playlist.txt`
   
   
Совет: Если вам часто нужно проигрывать музыку через терминал целыми папками, профессиональнее использовать `mpv`, который построен на библиотеках `FFmpeg`, но нативно поддерживает команды вида `mpv /путь/к/папке`.
