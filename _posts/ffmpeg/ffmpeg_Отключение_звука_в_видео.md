#### Отключение звука в видео

Для этого мы добавим простой тег: 

`-an`

`ffmpeg -i video_input.mp4 -an -video_output.mp4`

Примечание: 

Тег `-an`(auduo no) сделает все параметры звука для этого выходного файла бесполезными, поскольку в результате операции не будет звука.

Тег `-vn`(video no) без видео, только звук.
