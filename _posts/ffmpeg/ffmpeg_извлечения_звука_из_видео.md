В FFmpeg можно извлечь звук из видео — как без перекодирования, так и с перекодированием. Перед извлечением звука рекомендуется проверить исходный формат звука с помощью команды

- ffprobe (часть пакета FFmpeg)

#### Без перекодирования

Команда для извлечения звука из видео без перекодирования:

- ffmpeg -i input_video.mp4 -vn -acodec copy выходной_аудио.aac

Объяснение:

    -i input_video.mp4

    — указывает входной видеофайл.

    -vn

    — отключает запись видео, то есть будет обрабатываться только звук.

    -acodec copy

    — копирует аудиокодек без перекодирования, сохраняя исходное качество звука.

    выходной_аудио.aac

    — имя выходного аудиофайла, в котором будет сохранён извлечённый звук.

#### Важно: 

формат выходного аудиофайла должен соответствовать формату входного (например, если звук в формате AAC, нужно указать это - выходной_аудио.aac, выходной_аудио.vorbis, выходной_аудио.mp3)

#### С перекодированием

Команда для извлечения звука из видео с перекодированием (например, в формат MP3):

- ffmpeg -i input_video.mp4 outAudio.mp3

#### Объяснение: 

FFmpeg автоматически выбирает правильный кодек для преобразования аудиопотока и перекодирует его

#### Важно: 

из-за перекодировки процесс может занять некоторое время.
