***

Элемент или объект **minipage** очень удобно применять, когда нужно обособить какую-то часть текста. Для мини-страниц можно указать размеры — ширину и высоту, взять в рамку, сделать сноску, задать цветовой фон и т.д. Мини-страница может включать практические любые объекты.

  ## Синтаксис вызова

`\begin{minipage}[pos][height][valign]{width} 
 - `Содержимое страницы`
\end{minipage}`  
  
где:

- pos — вертикальная привязка к строке
- height— высота мини-страницы  
- valign — вертикальное выравнивание внутри мини-страницы  
- width — ширина мини-страницы  
  
Параметры, взятые в [ ] - не обязательные

Параметр **pos** может принимать значение t, c, b. 
Они означают, как мини-страница позиционируется относительно базисной линии строки, в которую она встраивается: 
t — верхняя строка мини-страницы встраивается в строку, 
с — мини-страница центрируется относительно базисной линии строки (по умолчанию), 
b — в строку встраивается нижняя строка мини-страницы.

Параметр **valign** может применяться только если задана высота мини-страницы. Этот параметр может принимать значение t, c, b, s, которые означают как текст выравнивается внутри заданной высоты мини-страницы: 
t — текст прижимается к верхнему краю, 
с — текст центрируется, b — текст прижимается к нижнему краю (по умолчанию), 
s — текст равномерно распределяется по высоте мини-страницы.  
## Аналог команды

Аналогом **minipage** является команда:  

`\parbox[pos]{height}[valign]{width}{text}`

Значения параметров такие же, как для **minipage**.

**parbox** удобнее использовать для небольших фрагментов текста.
