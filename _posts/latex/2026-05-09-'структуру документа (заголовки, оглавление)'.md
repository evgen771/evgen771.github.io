Для создания структуры документа в `LaTeX` используются иерархические команды. 

Система автоматически нумерует их и добавляет в оглавление.

### Основные команды иерархии

В стандартных классах (`article`, `report`, `book`) используются следующие уровни:

* `\part{Название}` — Часть (редко для статей)
* `\chapter{Название}` — Глава (только в report и book)
* `\section{Название}` — Раздел
* `\subsection{Название}` — Подраздел
* `\subsubsection{Название}` — Пункт подраздела
* `\paragraph{Название}` — Абзац с заголовком в строке

### Лайфхак: 

Если добавить `звездочку` (например, `\section*{Введение}`), раздел не будет пронумерован и не попадет в оглавление.

### Оглавление

Чтобы вставить оглавление, напишите команду:

`\tableofcontents`

Важно: 

Чтобы ссылки обновились, документ нужно скомпилировать минимум дважды.

### Пример шаблона документа

Вот как выглядит типичная структура научной статьи:

```latex
\documentclass[a4paper,12pt]{article}
\usepackage[utf8]{inputenc}
\usepackage[russian]{babel}

\title{Название вашей работы}
\author{Имя Автора}
\date{\today}

\begin{document}

\maketitle % Создает заголовок (название, автор, дата)

\tableofcontents % Автоматическое оглавление
\newpage % Переход на новую страницу

\section{Введение}
Здесь начинается текст введения.

\section{Обзор литературы}
\subsection{Отечественные источники}
Текст подраздела...

\subsection{Зарубежные исследования}
Еще немного текста...

\section{Методология}
Описание методов исследования.

\section*{Заключение} % Не нумеруется
Основные выводы.

\end{document}
```

Дополнительные фишки:

* `\appendix` — после этой команды все разделы будут именоваться буквами (Приложение А, Б...).
* `\listoffigures` — список всех рисунков.
* `\listoftables` — список всех таблиц.
