#### Сама преамбула:

```latex
\documentclass[a5paper,11pt]{report}
\usepackage[cp1251]{inputenc}
\usepackage[T2A]{fontenc}
\usepackage[english,russian]{babel}
\usepackage{amssymb,amsmath}
\usepackage[top=2cm,bottom=2cm,left=2cm,right=2cm]{geometry}
\setlength{\parindent}{0pt}  % убрать отступы в строках во всем тексте
\oddsidemargin=30mm   % При односторонней печати задает левое поле страницы.
\evensidemargin=30mm  % задают левое поле страницы для чётных страниц.   
\textwidth=110mm      % задаёт ширину тела документа. 
\topmargin=20mm       % задаёт верхнее поле страницы
\textheight=165mm     % задает высоту тела документа
\setlength{\parindent}{0pt}  % убрать отступы в строках во всем тексте
```

#### Пример создания документа:

```latex
\begin{document}

\textbf{Test} Преамбула стандартная.

\end{document}
```
