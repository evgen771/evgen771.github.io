Это можно элегантно обозначить, добавив ЧЕРНОВОЙ водяной знак на каждую страницу позади содержимого страницы. Для этого используйте пакет draftwatermark в преамбуле документа LaTeX:

- \usepackage{draftwatermark}

### draftwatermark

Пакет использует пакет everypage для вставки водяного знака на каждую страницу, поэтому этот пакет, возможно, тоже придется установить.

По умолчанию водяной знак добавляется на все страницы. Для вставки только на первую страницу используйте:

- \usepackage[firstpage]{draftwatermark}

По умолчанию используется текст водяного знака DRAFT. 

Чтобы вставить другой текст водяного знака (скажем BIO-HAZARD), используйте эту команду:

- \SetWatermarkText{BIO-HAZARD}

Чтобы задать другую яркость, используйте:

- \SetWatermarkLightness{0.5}

Значение яркости варьируется от 1,0 для белого до 0,0 для черного. По умолчанию оно установлено равным 0,8.

Масштаб текста по умолчанию равен 1.2. 

Чтобы изменить масштаб текста водяного знака, используйте:

- \SetWatermarkScale{4}

---

Большинство команд настройки можно использовать как в преамбуле, так и в теле документа.

### Что задается ТОЛЬКО в преамбуле

Сами опции при подключении пакета:

`\usepackage[firstpage, nostamp]{draftwatermark}`


* firstpage — выводить только на первой странице.
* nostamp — полностью отключить водяной знак.

### Что можно менять ВНУТРИ документа

Основные команды настройки (текст, масштаб, цвет) работают и после `\begin{document}`. Это полезно, если вам нужно изменить надпись в середине текста:

* \SetWatermarkText{...} — изменить текст.
* \SetWatermarkScale{...} — изменить размер.
* \SetWatermarkColor{...} — изменить цвет.

Важный момент: 

если вы измените параметр в середине страницы, он применится только к следующей странице. Это связано с тем, как `$\text{\LaTeX}$` формирует страницы в памяти.

### Как отключить/включить водяной знак «на лету»

Если вы хотите убрать знак с определенных страниц, используйте:

* \DraftwatermarkIt — включить (по умолчанию).
* \DraftwatermarkUnmark — выключить для текущей и последующих страниц.

```latex
Страница 1 (со знаком)
\newpage

\DraftwatermarkUnmark
Страница 2 (чистая)
\newpage

\DraftwatermarkIt
Страница 3 (снова со знаком)
```
