#### Включение–выключение нумерации страниц в LaTeX делается вот так:

- \pagestyle{empty} % нумерация выкл.
- \pagestyle{plain} % нумерация вкл.


Большего можно достичь, если использовать пакет `fancyhdr`.

Установить стиль нумерации для отдельной страницы можно командой

- \thispagestyle{стиль}


#### Начать нумерацию с другого номера:

- \setcounter{page}{3} % начать нумерацию страницы с номера три

В LaTeX есть как стандартная, так и пользовательская нумерация страниц. Это позволяет настроить стиль нумерации, указать начало отсчёта страниц или комбинировать несколько стилей. 

#### Стандартная

По умолчанию страницы нумеруются арабскими цифрами (1, 2, 3, ...). 

Отсчёт страниц начинается с первой страницы документа. Однако можно перезагрузить нумерацию в разных частях документа с помощью команды `\setcounter{page}{number}`. 

Здесь `number` — желаемый начальный номер страницы. 

#### Пользовательская

Изменить стиль нумерации можно с помощью команды `\pagenumbering{style}`. 

Аргумент `style` может принимать одно из значений:

- arabic — для стандартных чисел;
- roman — для строчных римских цифр (i, ii, iii, ...);
- Roman — для прописных римских цифр (I, II, III, ...);
- alph — для строчных букв (a, b, c, ...);
- Alph — для прописных букв (A, B, C, ...).

После подключения пакета `babel` с опцией `russian` можно использовать также значения `asbuk` и `Asbuk` для нумерации страниц русскими строчными и прописными буквами, соответственно. Количество нумеруемых страниц не должно превышать количество букв в алфавите. 

Можно комбинировать несколько стилей в одном документе. 

Например, в диссертации страницы до или после основных глав могут быть выполнены в римском стиле, а страницы основных глав — в арабском
