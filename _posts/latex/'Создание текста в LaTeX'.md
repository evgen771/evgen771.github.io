Создание текста в `LaTeX` — это процесс написания обычного текста внутри специальной структуры документа. В отличие от `Word`, вы сначала описываете логику (заголовок, автор, начало документа), а затем компилируете файл.

Вот основные шаги:

### Структура документа

Любой текстовый файл начинается с преамбулы и основного тела:

```latex
\documentclass{article} % Тип документа (статья)
\usepackage[utf8]{inputenc} % Поддержка кодировки
\usepackage[russian]{babel} % Поддержка русского языка

\title{Заголовок вашего текста}
\author{Ваше имя}
\date{\today}

\begin{document}
\maketitle % Команда для вывода заголовка, автора и даты

Здесь начинается ваш текст. Просто пишите предложения.
Для нового абзаца нужно оставить одну пустую строку.

\section{Первый раздел}
Текст внутри раздела. Вы можете использовать \textbf{жирный шрифт}, \textit{курсив} или \underline{подчеркивание}.
\end{document}
```

### Форматирование текста

* Жирный: \textbf{текст}
* Курсив: \textit{текст}
* Моноширинный (как код): \texttt{текст}
* Размер текста: \large, \huge, \small (ставится перед текстом).

### Списки

Для создания списков используются окружения itemize (маркированный) и enumerate (нумерованный):

```latex
\begin{itemize}
  \item Первый пункт
  \item Второй пункт
\end{itemize}
```
### Специальные символы

Некоторые символы в LaTeX имеют техническое значение, поэтому перед ними нужно ставить обратный слэш \:

* \$ — доллар
* \% — процент
* \& — амперсанд
* \_ — нижнее подчеркивание

### писать простой текст

Для написания простого текста в LaTeX тебе не нужно учить сотни команд. Достаточно знать «базовый каркас» и несколько правил оформления.

### Вот минимальный шаблон для русского языка:

```latex
\documentclass[12pt]{article} % Размер шрифта 12pt, тип документа — статья
\usepackage[utf8]{inputenc}    % Кодировка
\usepackage[russian]{babel}   % Поддержка русского языка

\begin{document}

\section{Заголовок}
Просто пиши текст, как в обычном блокноте. LaTeX сам позаботится о шрифтах и отступах.

\subsection{Подзаголовок}
Чтобы начать новый абзац, просто нажми \textbf{Enter дважды} (оставь пустую строку в коде). 
Одиночный перенос строки в коде LaTeX проигнорирует и склеит текст в одну строку.

\end{document}
```

### Главные правила для текста:

   1. Абзацы: Только пустая строка между блоками текста создает новый абзац.
   2. Кавычки: В русском языке принято использовать «елочки». В LaTeX их проще всего писать так: <<текст>>.
   3. Тире:
   * Короткое (дефис): -
      * Среднее (в интервалах цифр 1–10): --
      * Длинное (обычное тире): ---
   4. Шрифты:
   * \textbf{жирный}
      * \textit{курсив}
   5. Спецсимволы: Если нужно написать %, $, # или & просто как текст, ставь перед ними бэкслеш: \%, \$.

### Как сделать текст красивее?

Добавь в начало (до `\begin{document}`) эти строки для настройки полей:

`\usepackage[left=3cm, right=1.5cm, top=2cm, bottom=2cm]{geometry}`

Хочешь попробовать создать список или добавить картинку в твой текст?

### Как создавать списки

В LaTeX есть два основных типа списков: маркированный (точки) и нумерованный (цифры).

```latex
% Маркированный список
\begin{itemize}
  \item Первый пункт с точкой
  \item Второй пункт
\end{itemize}
```

```latex
% Нумерованный список
\begin{enumerate}
  \item Первый важный шаг
  \item Второй важный шаг
\end{enumerate}
```

### Как добавить картинку

Для работы с изображениями нужно добавить пакет `graphicx` в самое начало файла (в преамбулу).

Шаг 1: Добавьте пакет в начало файла:

```latex
\usepackage{graphicx}
\graphicspath{{images/}} % Необязательно: указывает папку, где лежат фото
```

Шаг 2: Вставьте код в текст:

```latex
\begin{figure}[h]               % [h] значит "поставить прямо здесь" (here)
    \centering                  % Центрирование
    \includegraphics[width=0.5\textwidth]{my_photo.jpg} % Имя файла и ширина
    \caption{Подпись к картинке} % Подпись под фото
    \label{fig:image1}          % Метка для ссылок на это фото в тексте
\end{figure}
```

### Итоговый «продвинутый» шаблон:

Скопируйте это в редактор, чтобы увидеть результат:

```latex
\documentclass[12pt]{article}
\usepackage[utf8]{inputenc}
\usepackage[russian]{babel}
\usepackage{graphicx} % Пакет для картинок

\begin{document}

\section{Мои планы на сегодня}
\begin{enumerate}
    \item Изучить основы LaTeX.
    \item Вставить красивую картинку.
    \item Написать список покупок:
    \begin{itemize}
        \item Кофе;
        \item Печенье.
    \end{itemize}
\end{enumerate}

\section{Визуализация}
Посмотрите на это изображение:

\begin{figure}[h]
    \centering
    % Замените 'example-image' на имя вашего файла, загруженного в проект
    \includegraphics[width=0.4\textwidth]{example-image} 
    \caption{Мой первый рисунок в LaTeX}
\end{figure}

\end{document}
```

### как делать ссылки в тексте

В `LaTeX` ссылки работают по принципу `«Метка» + «Обращение»`. 

Это очень удобно: 

если вы добавите новую главу или картинку в начало, номера в ссылках обновятся автоматически.

### Как создать ссылку

Для этого нужны две команды:

   1. \label{название} — ставится там, на что мы ссылаемся (внутри раздела, подписи к картинке или таблице).
   2. \ref{название} — ставится там, откуда мы ссылаемся.

### Пример с разделами

```latex
\section{Введение} 
\label{sec:intro} % Ставим метку
```

В разделе `\ref{sec:intro}` мы обсудим основы... % Ссылаемся (выведет номер раздела)

### Пример с картинками

Метку `\label` у картинок нужно ставить обязательно после команды `\caption`.

```latex
\begin{figure}[h]
    \centering
    \includegraphics[width=0.3\textwidth]{pic.jpg}
    \caption{Мой график}
    \label{fig:graph} % Метка здесь
\end{figure}
```

Как видно на рисунке `\ref{fig:graph}`, показатели растут.

### Ссылки на страницы

Если вы хотите сказать «см. на странице 5», используйте команду `\pageref`:

Подробности ищите на странице:

`\pageref{sec:intro}`

### Как сделать ссылки «кликабельными» (синими)

Чтобы в PDF-файле на ссылку можно было нажать и перейти к нужному месту, добавьте в начало документа (в преамбулу) пакет `hyperref`:

`\usepackage[colorlinks=true, linkcolor=blue]{hyperref}`

Важный нюанс: 

Чтобы номера в ссылках отобразились правильно, LaTeX иногда нужно скомпилировать дважды (в `Overleaf` это происходит автоматически).
