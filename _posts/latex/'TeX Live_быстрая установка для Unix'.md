## Вкратце: Unix (или что-то подобное)

Неинтерактивная установка по умолчанию на любых устройствах, кроме Windows:

    cd /tmp # выберите рабочий каталог по своему усмотрению
    Скачать: wget https://mirror.ctan.org/systems/texlive/tlnet/install-tl-unx.tar.gz
    золото: локон -L -o install-tl-unx.tar.gz https://mirror.ctan.org/systems/texlive/tlnet/install-tl-unx.tar.gz
    (или любым другим удобным для вас способом)
    zcat < install-tl-unx.tar.gz | tar xf - # обратите внимание на конечный - в этой командной строке
    cd install-tl-2*
    perl ./install-tl --no-interaction # от имени root или с правами root место назначения для записи
    # выполнение может занять несколько часов
    Наконец, добавьте /usr/local/texlive/YYYY/bin/PLATFORM в начало вашего файла ПУТЬ,
    например, /usr/local/texlive/2026/bin/x86_64-linux
